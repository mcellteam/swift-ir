• Ctrl + S - Save Project
  • Ctrl + Q - Quit Application
  • Ctrl + R - Refresh Tab
  • Ctrl + M - Enter/Exit Manual Alignment Mode
  • Ctrl + A - Align All
  • Ctrl + K - Skip Layer
  • ← (left arrow) - Previous Layer
  • → (right arrow) - Next Layer
  • ↑ (up arrow) - Zoom In
  • ↓ (down arrow) - Zoom Out
  • Ctrl + scroll - Zoom In/Out
